Introduction

In this lab, we will explore the basics of using the Turtle module in Python to create simple graphics. Turtle graphics allow us to provide an intuitive way to draw shapes by controlling a virtual turtle with movement and drawing commands. The primary objective of this lab is to learn how to initialize a turtle, customize its color and background, and move it to a suitable starting position. Once these steps are completed, we will use simple commands to draw a simple square. Through this lab, we will gain an understanding of how to manipulate the turtle's properties and control its movements, which forms the foundation for being able to create more complex graphics in the future. The lab will also introduce us to Python programming language and help us learn how use loops, functions, and coordinate- based movement, which makes it a valuable introduction to graphical programming in python.

Background

Python's turtle module is a modern implementation that is used to teach promgramming throguh visual representation. The module makes it easy for beginners to understand fundamental programming concepts such as loops, function, and coordinate- based movement. By using simple commands, the users are able to manipulate the turtle's position, direction, and appearance to create different shapes and patterns.

Methods / Algorithm

Below is our python code that imports the turtle package, initializes the turtle, sets the turtle's color and background color, and starts to move the turtle. 

!wget -O turtle_setup.py 'https://raw.githubusercontent.com/williamedwardhahn/fractal25/refs/heads/main/turtle_setup.py'
from turtle_setup import *
# Initialize the turtle
initializeTurtle()
showturtle()
color('blue')
bgcolor('white')

# Move to a suitable starting position
jump(200, 200)  # Adjust these values as needed to center the square
face(0)  # Ensures the turtle is facing up


forward(100)


# Display the drawing
show()

Conclusion

In this lab, we leveraged a JIT-optimized code in Python to generate both a broad and a zoomed-in view of the Mandelbrot set.
