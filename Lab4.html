Introduction

Fractals are geometric figures characterized by self- simularity and infinite complexity. They occur frequently in nature such as in plant growth patterns, mountain ranges, river networks, and weather systems. The purpose of this lab is to model a specific class of fractals known as binary fractal trees. We will do this by using recursive programming and turtle graphics in Python. The primary objective is to develop a visual understanding of recursion and its applications in fractal geometry. By constructing a branching structure where each branch recursively splits into two smaller branches, we simulate the recursive nature of many naturally occuring systems. This lab also provides an opportunity to examine how changes in recursion depth, branch angle, and reduction ratio influence the resulting geometry.

Background

Fractals are mathematical sets that exhibit a repeating pattern at every scale and are often described as infinitely complex. A fundamental property of fractals is self- similarity, meaning parts of the structure resemble the whole. In computer science, fractals are frequently generated using recursive algorithms, where a function calls itself to repeat a pattern at smaller and smaller scales. One classic example of a fractal is the fractal tree, which visually mimics the way tree branches subdivide. This structure can be constructed using turtle graphics- a Python library used for vector drawing based on a cursor that can move and draw lines on a canvas. By recursively reducing the branch length and altering turning angles, the turtle draws complex structures that demonstrate fractal behavior. This lab gives a hands on approach to learn how the power of recursion is simulating real- world occurences and provides a visual medium for understanding mathematical abstraction.

import random

# Function to draw a fractal tree using recursion
def fractal_tree(length, depth):
    if depth == 0:
        return
    else:

        forward(length)
        left(30)

        fractal_tree(length * 0.7, depth - 1)

        right(60)

        fractal_tree(length * 0.7, depth - 1)

        left(30)
        backward(length)

def build_tree(branch_length, shorten_by, angle):
    if branch_length > 3:

        forward(branch_length)

        new_length = branch_length - shorten_by

        left(angle)

        build_tree(new_length, shorten_by, angle)

        right(angle * 2)

        build_tree(new_length, shorten_by, angle)

        left(angle)

        backward(branch_length)
